# API Testing con Postman y Supertest

## Descripción

Este documento presenta el desarrollo y validación de una API utilizando **Postman** y **Supertest**. Se realizaron pruebas manuales y automatizadas para verificar el correcto funcionamiento de los endpoints, validar respuestas HTTP y asegurar la calidad del software.

## Objetivos

- Comprender el funcionamiento de las API REST.
- Realizar pruebas manuales mediante Postman.
- Implementar pruebas automatizadas utilizando Supertest.
- Verificar códigos de estado y respuestas de la API.
- Aplicar buenas prácticas de testing en aplicaciones backend.

## Herramientas Utilizadas

- Node.js
- Express.js
- Postman
- Supertest
- Jest

## Contenido

1. Introducción a las API REST.
2. Configuración del entorno de desarrollo.
3. Pruebas manuales con Postman.
4. Pruebas automatizadas con Supertest.
5. Casos de prueba implementados.
6. Resultados obtenidos.
